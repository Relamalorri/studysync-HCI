# Evaluation

## Evaluation Tasks

1. Sign Up / Log In
2. Add a new course with color labeling
3. Add an assignment with deadline and priority level
4. Check today's tasks and upcoming deadlines from the dashboard
5. Use the calendar to view weekly and monthly deadlines
6. Mark an assignment as completed
7. Set a reminder notification for an upcoming assignment
8. View daily tasks and upcoming deadlines

---

## Participants

10 undergraduate students from the Faculty of Computing and Information Technology at King Abdulaziz University. Selected because they manage multiple courses, assignments, and deadlines each semester — the exact target user of StudySync.

---

## Testing Environment

Conducted on campus in quiet settings (study rooms, libraries, lounges) using participants' personal phones with the Figma prototype. Each participant was briefed before testing; the facilitator observed interactions, measured task completion time, and noted any navigation difficulties.

---

## Usability Results

| Task | Avg Time | Errors |
|---|---|---|
| Sign Up / Log In | ~14s | 2/10 participants |
| Add Course with Color | ~20s | 2/10 participants |
| Add Assignment | ~37s | 3/10 participants |
| View Calendar Deadlines | ~11s | 0/10 participants |
| Mark Assignment Complete | ~6s | 0/10 participants |

---

## Usability Goals

- **Effectiveness** — Users completed all tasks successfully
- **Efficiency** — Most tasks completed in under 20 seconds
- **Learnability** — Interface was intuitive with no training required
- **Satisfaction** — Users felt organized and less stressed after using the app

---

## Key Findings

- Viewing deadlines and marking tasks complete were the fastest and most intuitive tasks
- Adding assignments took longer due to multi-step input (name, type, course, priority, deadline)
- Color-coded courses were highly appreciated for reducing cognitive load
- Minimalist design and clean navigation received positive feedback
- Several participants noted that reminders would significantly reduce academic stress

---

## Conclusion

The StudySync prototype successfully supports students in managing academic responsibilities efficiently while delivering a positive, organized user experience.
