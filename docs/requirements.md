# Requirements

## Target Users

University students (Bachelor's and Master's) who are digitally fluent, rely heavily on smartphones, and need a dedicated academic task management tool.

## Context of Use

- **Physical:** Used on personal smartphone — commuting, between classes, at home
- **Social:** Individual use — each student manages their own workload
- **Technical:** iOS and Android; offline mode for basic features
- **Sessions:** Designed for short sessions under 2 minutes

---

## Interface Requirements

### Simple and Structured Navigation
Clear navigation enabling students to reach courses, assignments, and schedules in minimal steps.

### Course-Based Visual Organization
Color-coded labels for each course to help students distinguish between subjects at a glance.

### Unified Academic Dashboard
Home screen showing all tasks in one place — prioritized by deadline and urgency.

### Calendar and Deadline Management
Weekly and monthly calendar views with easy switching between them.

### Smart Reminder and Notification System
Push notifications for upcoming deadlines that are visible without disrupting the user experience.

### Quick Task Creation and Editing
Add, edit, complete, or delete tasks within seconds — speed is essential.

### Readable and Minimalist Interface
Clean fonts, well-structured layouts, no visual clutter.

### Progress Tracking and Motivation
Progress indicators per course plus daily motivational quotes.

### Consistency and User Feedback
Consistent colors, icons, and navigation throughout. Immediate feedback on every user action.

## User Research Findings

Data collected from 11 participants via Google Forms:

| Finding | Result |
|---|---|
| Use productivity tools | 81.8% yes |
| Biggest challenge | Forgetting deadlines (54.5%) |
| Most wanted feature | Simple clean design (91.8%) |
| Device preference | Smartphone (45.5%) |
| Task planning time | 5–15 minutes weekly (27.3%) |
| Miss deadlines per week | 1–2 times (27.3%) |
